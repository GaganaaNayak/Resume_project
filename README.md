# Personal Resume Website

A modern multi-page resume and portfolio website built using HTML, CSS, JavaScript, and jQuery.
Includes pages for Home, About, Projects, and Resume, with a clean UI and responsive design.

# Project Structure

Resume/index.html
Resume/about.html
Resume/projects.html
Resume/resume.html
Resume/assets/css/style.css
Resume/assets/js.main.js
Resume/assets/images/(profile.jpg, project screenshots, etc.)
